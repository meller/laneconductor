# Track 015: Track Conversation Inbox

**Lane**: done
**Lane Status**: success

## Problem
Currently, users must open each track individually to see if there are new messages or AI responses. There is no central place to manage all active conversations.

## Solution
A global "Inbox" view that aggregates all track conversations, highlighting unread or unreplied messages, with deep links to jump directly to the relevant track and tab.
